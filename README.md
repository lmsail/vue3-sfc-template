# vue3-sfc-template
 
vue3 + vite + vuex + vant 脚手架


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
npx vite --port=4001
```

### Compiles and minifies for production
```
npx vite build  

# 二级目录部署
npx vite build --base=/web/ 
```
